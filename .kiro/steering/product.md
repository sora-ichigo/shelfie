# Product

本ドキュメントは、Shelfie のプロダクト方針を定義する。

## プロダクトビジョン

**「気持ちいい読書管理 + 友達と本棚を共有する」アプリ**

気になった本を即記録し、友達と読書体験を共有できる場所。

## コアバリュー

> あなたの本棚を、手のひらに。友達の本棚も。

- **Why（存在意義）**: 気になった本を即記録し、友達と読書体験を共有できる場所
- **How（差別化）**: モダン UX × 日本語書籍 × プライベートな友人つながり
- **What（機能）**: 読書管理、友達の本棚閲覧、AI レコメンド（将来）

## 読書サイクル

Shelfie が支えるユーザー行動の全体像。

```
出会う → ためる → 読む → 残す → 見返す → また出会う
```

「出会う」の主要エントリーポイントは**友達の本棚**。友人のおすすめは AI レコメンドより信頼性が高い。

## プロダクトレイヤー

```
土台: シンプルで気持ちいい読書管理（Phase 1）
  ↓
モート: 招待制の友人ネットワーク（Phase 2）
  ↓
拡張: オープンなフォロー制（Phase 3）
  ↓
収益: Pro サブスク / アフィリエイト / AI レコメンド
```

## ソーシャルの設計方針

- **LINE 型から始めて Instagram 型に拡張する**
- 招待リンク・ユーザー検索・QR コードで友達とつながる
- 本棚はデフォルト非公開。つながった友達にだけ見える
- おすすめユーザーやランキングは設けない
- グループ機能は作らない（グループは過疎って終わるため）
- Phase 3 で本棚の公開オプションを追加し、フォロー制に拡張

## ターゲットユーザー

| Phase | ターゲット | 訴求 |
|---|---|---|
| Phase 1 | 既存アプリに不満を持つ読書家 | シンプル × モダン UI で乗り換え |
| Phase 2 | 友達と本の話がしたい読書層 | 友達の本棚が見える体験 |
| Phase 3 | SNS で本に触れている若年層 | 本棚公開 + AI レコメンド |

## マネタイズ

ソーシャル機能は成長エンジン。収益エンジンではない。

- **基本無料**: コア機能（登録、管理、検索、リスト、友達つながり）はすべて無料
- **Shelfie Pro（サブスク）**: 高度な読書統計、AI レコメンド、データエクスポート等
- **アフィリエイト**: 友人おすすめ経由の書籍購入（コンバージョンが高い）
- **方針**: まず使われるものを作る。マネタイズは定着してから設計する
